
This project is a responsive website for a fictitious company called Accessible Design Co.
The goal of this project is to practice semantic HTML, responsive layout using Flexbox and CSS Grid and basic web accessibility.

The website includes a header with navigation, a hero section, an about section, services, a contact form, and a footer.  
Accessibility features like proper labels, ARIA attributes, and keyboard-friendly navigation are included.

How to View or Run This Project

1. Download or clone this repository to your computer.
2. Open the folder in your code editor (for example, VS Code).
3. Find the file named `index.html`.
4. Double-click `index.html`or right-click and open it in a web browser.
5. The webpage will open and you can view it in desktop or mobile screen sizes.


Project Purpose 
- How to structure HTML using semantic elements
- How to create responsive layouts
- How to make a webpage more accessible for all users
